###Definitions:
- Pattern:
    - it solves a problem
    - it is a proven concept
    - the solution is not obvious
    - it described a relationship
- Prototype:
    - an encapsulation of properties that an object links to


###Types of patterns

- #### Creational
    - Constructor 
    - Module
    - Factory
    - Singleton
    
- #### Structural
    - Decorator
    - Facade
    - Flyweight
    
- #### Behavioral
    - Command
    - Mediator
    - Observer
